# 读取 & 存储



```python
import torch
# 保存整个网络
torch.save(net, PATH)
# 保存网络中的参数, 速度快，占空间少
torch.save(net.state_dict(),PATH)
#--------------------------------------------------
#针对上面一般的保存方法，加载的方法分别是：
model_dict=torch.load(PATH)
model_dict=model.load_state_dict(torch.load(PATH))
```



结合 tensorboard 

```python
from torch.utils.tensorboard import SummaryWriter

writer = SummaryWriter('logs')

g_loss = torch.load('debug/generator.losses')
for step in range(len(g_loss)):
    writer.add_scalar('Loss/g_loss', g_loss[step], step)
```
